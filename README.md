# Tarea_02 por Álvaro Jopia

En este repositorio trabajé ejercicios enfocados en entender la lógica de programación. Se toman datos, se aplican condiciones y se clasifican resultados.

---

## Problema 1: Clasificación de temperaturas

Se recorrió un rango de números (de -30 a 50) y se clasificaron en tres categorías: muy frío, templado y muy caluroso.
La clave fue usar condiciones ordenadas para dividir los valores según su magnitud.

---

## Problema 2: Números del robot

Se evaluaron los números del 1 al 100 para identificar cuáles son divisibles por 5.
Además, se agregó una segunda condición para distinguir los que también son divisibles por 10, asignándoles un mensaje distinto.

---

## Problema 3: Puntaje gamer

Se pidieron dos valores al usuario, se sumaron y el resultado se clasificó en tres niveles: principiante, intermedio y pro.
Aquí lo importante fue trabajar con rangos y ordenar correctamente las condiciones.

---

## Problema 4: Perfil musical

Se analizaron tres variables (edad, gusto por la música y gusto por bailar) para definir un perfil.
Se utilizaron condiciones combinadas para evaluar distintos escenarios y asignar un resultado según los datos ingresados.

---

## Conclusión

Los ejercicios muestran una misma estructura: recorrer o recibir información, aplicar reglas y obtener una clasificación.
Así pude entender las bases de la programación.
